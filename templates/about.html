About This Project

AI-Powered Sign Language Translator — UG Final Year Project

Project Overview

This is an AI-powered bidirectional sign language translation system. It bridges the communication gap between sign language users and non-signers by providing two core capabilities: real-time sign-to-text translation using computer vision and deep learning, and text-to-sign conversion using 3D animated hand gestures.

The system supports the American Sign Language (ASL) alphabet (A–Z), digit gestures (0–9), and common word-level signs including Hello, Hi, Good Morning, Thank You, Sorry, Please, Yes, and No. A total of 44 sign classes are recognized by the AI model.

System Architecture

Sign → Text Pipeline: The webcam captures live video frames, which are processed by Google's MediaPipe Hands to detect 21 hand landmarks with 3D coordinates. These 63 features (21 landmarks × 3 axes) are fed into a trained neural network that classifies the hand shape into one of the 44 sign classes. A sentence builder module with debouncing logic accumulates detected signs into readable text, automatically inserting spaces when the hand is removed from view.

Text → Sign Pipeline: User-entered text is preprocessed and checked against the common word gesture database. Known words are rendered as complete sign animations, while unknown words are finger-spelled letter by letter. The 3D hand model is rendered using Three.js with WebGL, featuring 21 animated joint nodes connected by bone segments, with smooth interpolation between sign poses.

AI Model Details

The recognition model is a Multi-Layer Perceptron (MLP) neural network built with scikit-learn. It takes the 63-dimensional landmark feature vector and passes it through a deep architecture of hidden layers (512→256→128→64 neurons) with ReLU activation. The pipeline includes StandardScaler for feature normalization and uses the Adam optimizer with adaptive learning rate and early stopping.

The model is trained on a generated dataset of 600 augmented samples per class (26,400 total), created from canonical ASL hand landmark positions with random noise, scaling, translation, and rotation augmentation.

Key Features

Real-time inference — Predictions at ~10 FPS with sub-200ms latency. Sentence building — Automatic space detection and word recognition. Auto-training — Model trains automatically on first launch. Offline capable — Works without internet after setup. 3D animation — Smooth hand gestures rendered in WebGL. Single-file backend — Entire Python backend in one file.
